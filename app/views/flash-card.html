Interaction principles

An interruption card should be used to draw the attention of the user to some important information specific to that point in their journey.

Cards should have headings. You may want to hide the heading but it's generally simpler not to.

One thing per card. If the user needs to do more than one thing, use extra cards.

Use larger text (24px) than in the rest of the journey.

Use a “Continue” button to advance to the next card or page.

Don't use form elements (except buttons) within cards. Form elements will add extra complexity and you will need to deal with error states.
