Modal window

Creating focus on a single interaction.

  • Usage: internal
  • Status: Alpha
  • Last updated: 17 March 2017

Interaction patterns

  • the modal should focus on a single task
  • consider an alternative approach to modals on mobile
  • could the interaction be handled on another screen

Content patterns

  • content that opens the modal should explain to the user what will happen
  • the content of the modal should be focused on the single task and continue from the link/page
  • include heading or title

Inclusive patterns

  • modals do not work well on mobile
  • screen readers do not automatically focus on modals which makes interactions confusing

User research

  • encourages focus
  • complicated interactions within the modal dissorientate
  • multiple modals create confusion
  • test this version (CSS only) with form content

Services using this pattern

  • Application
  • Employers Checking Service
  • Passport renewals (timeout)
