Why have we created this pattern?

No GDS pattern is available.

Interaction principles

The modal window should focus on a single task.

Before using a modal window, consider whether the interaction could be handled on another screen.

Content principles

The content that opens the modal window (for example, anchor text) should explain to the user what will happen.

The content of the modal window should be focused on the single task and should not add unnecessary detail.

The content should be consistent with the referring page so that the user is not confused.

The modal window should have a clear main heading.

Accessibility principles

Modal windows do not work well on mobile. Consider using an alternative approach.

Screen readers do not automatically focus on modals, which makes interactions confusing.
