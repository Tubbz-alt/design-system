Uploading files

An interaction that allows a user to upload one or more files.

  • Usage: internal/external
  • Status: experimental
  • Last updated: 10 October 2017

Upload a single file

Upload multiple files

Guidance

It can be used:

  • to upload single files
  • to upload multiple files
  • for a range of file types

It should not be used:

  • in some internal services where there is a need for bulk uploading

The upload pattern should:

  • replay the file clearly to reduce user error
  • have flexibility so users can change their minds
  • allow the user to skip the process if allowed
