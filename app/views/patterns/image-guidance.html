Guidance images

Using images can help explain content to users.

When to use this pattern

Only use images when research identifies a user need. Used correctly, images can help users to understand:

  • a new or unfamiliar concept
  • complex guidance when completing a task
  • how to find a reference number on a document

Examples

Services using this pattern

  • Passport renewals
  • Right to work
  • Registered Traveller
