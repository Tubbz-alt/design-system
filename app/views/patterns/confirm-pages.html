Why have we created this pattern?

The GDS pattern uses a lighter green which does not meet accessibility requirements for some smaller text.

The incomplete version was created following user evaluation where users were confused as to wether the task was complete or not.

In general, the variations have been designed to make clearer at what stage the application now is and next steps for the user.
