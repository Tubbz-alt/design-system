Postcode lookup

A way for users to find there address using a postcode.

  • Usage: internal/external
  • Status: experimental
  • Last updated: 10 December 2017

Guidance

It can be used:

  • to find a single address
  • to add an address manually

It should not be used:

  • if the address is needed as structured data

The postcode lookup pattern should:

  • replay the address options clearly
  • give the user flexibility to add an address in different ways
