Interaction principles

  • only one nav item can be active at a time.
  • the active nav item should be distinguishable from the inactive.
  • inactive tabs should have a hover state.
  • focused nav items should be consistent with other focus states.
  • don’t use animation unless there is a proven need to do so.

Content principles

  • the label should reflect the thing you need do at the destination, not a description of what it is.

Accessibility principles

  • the active nav item should be labelled as active in the code.
  • the active nav item should be distinguishable from the inactive, using more than colour alone.
