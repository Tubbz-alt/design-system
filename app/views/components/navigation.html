Interaction principles

Only one nav item can be active at a time and an active item should be distinguishable from inactive items.

Inactive items should have a hover state.

Focused nav items should be consistent with other focus states.

Content principles

The label should reflect the thing that the user will do at the destination, not describe the destination.

Accessibility principles

The active nav item should be labelled as active in the code.

Colour should not be the only way to distinguish active from inactive nav items.
