Why create this pattern

The current GOV.UK component isn't consistent with link styling and icons across Government. Research has found some users are anxious using the back link. Confidence might increase by being more descriptive about the page it would naviate back to.

Interaction principles

  • Always include a back link on question pages.

Content principles

  • Not all page titles will be suitable for the back link.

User research

  • Some users thought using the previous back link would undo work. And weren't confident where the back button would take them.
