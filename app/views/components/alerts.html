When to use this component

Use alerts to inform users that something important has happened. Either an action they need to take or to notify them of a success/failure.

When not to use this component

Only use when you have identified that users need alerting to certain content.

User research

Passport renewals found this component worked better then the confirmation pattern for a payment success page. They observed users engaged with the next steps on the page, rather than thinking they were finished.
