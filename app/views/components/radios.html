When to use this component

Use this component on government staff and caseworking services. It can be a useful alternative to the larger GOV.UK radios when there is more then one thing on a page.

A common use would be filtering search results.

When not to use this component

On public facing services. And as part of the 'one thing per page' pattern.

User research

A number of services are using this pattern in the Home Office. From the research we have, we haven't identified any issues with the size or usability of the smaller radios.

Component needs further research on mobile devices.

Services using this pattern

  • DAP
  • Resettlement services
  • General Aviation
  • ASRU
  • Application platform
  • Data services
