Guidance images

Using images to better explain content to users.

  • Usage: internal/external
  • Status: experimental
  • Last updated: 25 August 2018

When to use this component

Most content will not need an image. If users are struggling and you can see a real user need, consider introducing an image to see if it helps. Examples where an image might help users:

  • to explain a new concept to users
  • for more complex guidance when completing a task
  • finding a reference number on a document
