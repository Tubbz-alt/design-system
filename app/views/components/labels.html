When to use labels

Labels help users identify grouped content. They are for quick referencing so should change as the status of a thing changes.

How to use labels

Labels should be single words and use plain English to describe the grouping. We recommended that labels are lowercase.

Colour combinations should meet the minimum contrast levels for WCAG AA.

User research

Adding labels to everything is counterintuative
